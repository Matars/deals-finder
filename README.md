# deals-finder

Self-healing second-hand deal finder powered by [Hermes Agent](https://github.com/nickthetasmc/hermes-agent).

**Scrapes tech deals from vendor websites, estimates discounts, publishes to GitHub Pages. When scrapers break, Hermes auto-repairs them.**

## Quick Start

```bash
# 1. Clone
git clone https://github.com/Matars/deals-finder.git
cd deals-finder

# 2. Setup (interactive wizard)
pip install pyyaml
python setup.py

# 3. Generate scrapers (requires hermes)
python setup.py --generate

# 4. Test
python scrape.py --check    # verify scrapers load
python scrape.py            # run a full scrape

# 5. Enrich + publish
python scraper/enrich_json.py docs/latest.json
git add -A && git commit -m "update deals" && git push
```

```
└─────────────┘     └──────┬───────┘     └──────────────┘     └──────────┘
                           │
                    ┌──────▼───────┐

                    │   by hermes) │
                    └──────┬───────┘
                           │ error?
                    ┌──────▼───────┐
                    │  self-heal   │
                    │  (hermes     │
                    │   repairs)   │
                    └──────────────┘
```

## The Magic: Self-Healing

When a vendor changes their website DOM, scrapers break. Instead of crashing silently:

1. `scrape.py` detects the failure and logs to `docs/.scrape_errors.json`
2. Run `python setup.py --heal` (or use `--auto-heal` flag)
3. Hermes agent:
   - Reads the broken scraper code
   - Navigates to the vendor site with browser-harness
   - Analyzes the new DOM structure
   - Patches the scraper with corrected selectors
   - Tests the fix
4. Scraper works again

```bash
# Manual healing
python setup.py --heal

# Automatic healing (after each scrape)
python scrape.py --auto-heal
```

## Configuration

`config.yaml` defines your setup:

```yaml
user:
  language: sv        # your language
  country: SE         # your country
  currency: SEK       # output currency

vendors:
  - name: blocket
    url: "https://www.blocket.se/recommerce/forsale/search?category=0.93&q={query}"
    enabled: true

categories:
  - name: gpu
    icon: "🎮"
    queries:
      - gpu
      - grafikkort

self_heal:
  enabled: true
  max_retries: 2
```

## Adding a New Vendor

### Automatic (requires hermes)

```bash
# Edit config.yaml to add the vendor, then:
python setup.py --generate
```

Hermes will navigate to the site, analyze the DOM, and write a scraper module.

### Manual

Create `scrapers/<vendor>.py`:

```python
import sys, os
sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))
from scrapers.base import ScrapeResult

def scrape(query: str, category: str) -> ScrapeResult:
    items = []
    # ... your scraping logic ...
    return ScrapeResult(items=items, vendor="myvendor", query=query)
```

## Architecture

- **`scrape.py`** — Runner. Loads config, dispatches to scrapers, collects results.
- **`scrapers/base.py`** — Interface definition + dynamic module loader.
- **`scrapers/<vendor>.py`** — One file per vendor. Generated by hermes or hand-written.
- **`enrichment/`** — Price estimation module. Adds original prices and discount %.
- **`setup.py`** — Interactive setup + hermes integration for scraper generation/repair.
- **`config.yaml`** — User configuration.

## Dependencies

- Python 3.11+
- pyyaml
- [browser-harness](https://github.com/browser-use/browser-harness) (for scraper generation)
- [Hermes Agent](https://github.com/nickthetasmc/hermes-agent) (for scraper generation + self-healing)
